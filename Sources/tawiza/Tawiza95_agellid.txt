 uïïun 95, krayur 2005 (Mars 2005) Tamazight Agellid Cacnaq. Yura t: Nadir Amazigh. Jar Imazighen d Imizvriyen iqburen azday (relation) d useqqen zegh talji n umezruy, azday n sin n udmawen: udem n umennugh d udem n talwit. llant titgarin (les documents), tasregta (la gravure) tapipugrafit d unugh (dissin) af isemdval yemmalen imenghan yemsaren nger Ifiroawniyen t tiwsatin (les tribus) n Imazighen ilibiyen "Mashaoush" izedghen imir nni ataram (ouest) n wasif n Nnil amek id yenna Pirudut (Hirodote). Asekfel d akw isiggilen di tagherma n Imizriyen iqburen ssenzan (démontrer) belli tilin (existence) n Imazighen g wakal Afiroawni llan zi zik deg w akud igguj 5000 iseggusa. Ad nini belli d sin igherfan icawraren ingadda (égal) azemz (date) nsen u tugh d igduden idduklen di cra n tighawsiwin, amedya iles d useged (ajjed) u rnu gher aya ad ten naf ula d ijentvadven (les voisines) n tilisa. Taghremt tamenzuyt n Imazighen yeqqnen amsawadv aked Imizvriyen iqburen nettat t taghremt lli umi qqaren “Libu”, tugh illan di tmurt n Libya n imir a, u ssin dewlen Imazighen qqaren asen Ilibiyen d merra timura n Tamazgha tughal “Libye”, ladgha (surtout) ar Igrigiyen iqburen. Deg usenti n udabu n tawja (tawacult) tis tam d merawt (18) din ssentan Imazighen Ilibiyen ttadsen (rapprocher) ibuda (limites) n Mizvra n umalu ssuturen tazeddight ar idis n wasif n Nnil, lughddin (imir nni) dvran jar asen imsenghan lli ghur smattarent (indiquer) titgarin. Agellid Ramsis wis kradv ur yessawedv ad yesbed akw timdvugin (les attaques) n Imazighen armi gguten guda saman xef wasif n Nnil u ssentan ula d netnin ad ilin d iserdasen n tawacult tis simrawt (20) t tutvfa n tiwuriwin di tdabla (administration). deffer tamettant n ugellid aneggaru “Busennes” wis sin n tawja tis simrawt d yut (21) din teghlid tanbadvt ar Imazighen ghef ixef nsen amizer (chef) Cacnaq amezwaru amessenti (le fondateur) n tawacult Tamazight tis sinmrawt d snat (22) di 945 zdat talalit u yefren “Bubastis” ttamannaght. Di turemt (temps) n udabu n ugellid Alibi Amazigh Cacnaq, isdukkel Mizvra lli tugh innebdvan af sin igezzumen (parties) Thebe d Tanis. af ufus is issuksed tamurt zeg tilufa irrat id ar talwit s useggem (l’amendement) n tighawsiwin n timetti d usnarni n tsertit, iserdasen d aseged “Amun”. Cacnaq issanef amezruy itnagan belli d agellid lli ur dar s amedya di akw tiwaculin itvfen adabu g Mizvra. Zi Bubastis tamurt n Jacun amek tetwassen ghar Udayen ifta (partir) Cacnaq d iserdasen is azveggu ar umenzvaw n Asya, kkin af Sina udfen Ghazza ssin hewsen igherman nnidven n Falastvin. issenger iserdasen Udayen mgal tasertit lli ur idga (accepter) issefru merra tilufa din u yeqqim Cacnaq d azammul n talwit. Deg udlis aghudid “Bibel” tasarnatv (1 Igelliden 14,25-26) itwabder di s Cacnaq lemmi ikcem ar Jurzalam. 25 aseggwas wis semmus n ugellid Roboam, cacnaq, agellid n Mizvra izvdvem mgal Jurzalam.26 iksi akw igarujen n taddart n Yahve d igarujen n ibergemmi ugellid: iksi akw. amek iksi akw araren n uregh igga Suliman (25 La cinquième année du roi Roboam, Chichaq, roi d’Égypte monta contre Jerusalem. 26 Il prit les trésors de la maison de Yahve et les tresore du palais du roi: il prit tout, comme il avait pris tous les boucliers d’or qu’avait faits salomon) 2002 Tawiza. .
