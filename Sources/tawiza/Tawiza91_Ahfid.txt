 uïïun 91, yemrayur 2004 ( 2004) Tamazight Ayyuz i Tamesmunt n Numidya di Pulanda Hassan Ahfidv (Aliman) Tamddit n 9-10-2004 tesmun anegh tamesmunt n Numidia di Pulanda-Venlo- deg ij n wurar d amazigh d ameqran s tactut -Lmunasaba– n usggûas amezwaru n trurit nnes. Usin d ghar s attvas n imazighen i texsen tcuni d reghnuj d umezgun amazigh, d wattvas n imeghnas i d yusin s yisem n umussu adelsan amazigh zi Pulanda d Uliman d Danmark (mass Mtalsi muhamed d mass Munir). Tidet tughat d ict n tmaddit tecna tehla s wattvas s ij n wudem itfuh bezzaf. Ghar umezwaru -awarni warar n umghar n tmesmunt iqad d ayes marra win i d yusin ghar sen – tassenta tasedvma n tayri n ighennijen n tmazight iccuren s refwah n idurar n Atlas amezvyan, ighennijen n tayri d tmexsa d arridva s wudem nnes yizdigen. Awarnas mmas Aloisati obdrrahman issiwer x tmazight d ubrid n umussu adelsan amazigh zeg isggusa n 56 ar 2001, asggûs n trurit n anelmas n tmazight di yarbatv d tghuri n tmazight deg ugherbaz mamec tessiwer ict n tanujit tusd akid s - s yisem n IRKAM- x marra min itwaggen d idisen i wesghar n tmazight s uskkil n tifinagh, tennurzvem tewwurt i wattvas n iseqsiten i tugha illan deg uraq nsen. Teodu tasedvma n Isri zi Pulanda, d tasedvma n yitran s ighennijen nnes icawwaren –waxxa nezvwa yaman, tcuq d tudart inu, yuc icem babam,… Tasedvma n umezgun” Imddukar” tesqas anegh s ij ughezdis d amezvyan zi tmezgunt nsen – jar remrac d reprac-,. Azrawi –amnus- ighar d x af negh ca n tqessisin nnes itfuhen s iwaren n tmazight icwwaren. Ghar unggar – ghar tagara - Yuri d ucennay ameqran i tugha trajan marra, netta d Walid Mimoun s reghnuj nnes igha yiqqimen arecmen di tawengimt nnegh, Dcar inu, d urinu d remwaj, ayt ma imezvyanen d wattvas n ighannijen icnan. Ad tqad i tmesmunt n Numidya x wurar a yimgharen n tmazight i tegga . Tudart d tcuni i tamazight. (Ass n 13-10-2004/2954) 2002 Tawiza. .
