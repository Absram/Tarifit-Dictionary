 Uttvun 86, Sedyur 2004 (Juin 2004) Tamazight Adllis amazigh s tutlayt n talimant Yura t: Hassan Ahfidv (Aliman) Iffegh d ij n udllis icna s wattvas, day s bezzaf n useqdec i ghad narni ghar twngimt nnegh. Adellis a s yires n tlimant n massa Gabriele Kratochwil -qqarnas "amussu amazigh di Lmughrib", amezruy n webrid nnes x wudem n tanttit tamazight-jar asgwas n 1912-1997-yus d di 489 n tefray, atig nnes 49 euro. Trur d tamarirt massa Gabrielle Kratochwil asgwas n 1968 di tendint n Kölen -aliman- Teghra tusna n Islam, d tusna n tasertit, di tesdawin n Bonn d qahira-misra-, tassenta ad tarzu deg iyyar n umussu amazigh di Lmughrib ameqran asgwas n 1995, nttat d tamarzzut tamezwarut deg uliman iyggin ij n tighuri tadimuqratit s wudem d uxarres d amusnawi-scientifique- iwmussu amazig di Lmughrib ameqran, tugha t zeg iwdan iyardfen iyddaren agraw amazigh amezwaru di tafira-kanarya, tura attvas n imeghraden deg ighmmisen d tisghunin n talimant, tegga attvas n imsagaren x min yimsaren di tafira. (Ass n 1-5-2004-2954) 2002 Tawiza. .
