 uïïun 91, yemrayur 2004 ( 2004) Tamazight Aslmd n tmazivt niv tiknnaä tixatarin n timaä (L'enseignement de tamazight ou le grand leurre du siècle) Yura t: mupmmd budhan Qao izvnan (critiques) i ytwaggn i wsinag agldan n tussna n tmazivt (Irkam) tuva ottun (zrryn) dg udm n ulmmiç (acquis) n takraäit (le fait d'être obligatoire) n uslmd n tmazivt mani ur tiwiä ula d dzayr i d anv yizzarn dg uslmd n tmazivt. aïïas (cigan, bahra) n imxumbal (problèmes, difficultés) d wktam (imperfection, insuffisance) ^^çlyn (distinguer) aslmd n tmazivt dg wsgåas amzwaru. Maca nenna manayad d agmawi (naturel) itssn it kuy (chaque) usnfar (projet) dg usnti (début) nns. Nenna is aktamn nni ur tvimin dg usgåas wis sin n uslmd n tmazivt. Ha asgåas wis sin iddm (issnta) nnhla (asl, mbla = sans) ad in ilint ca n tareççimin (solutions) i tmukrisin i itbddan dg ubrid n uslmd n tmazivt. S umngal, (au contraire), tisiwav (entraves) n usmd ad mmarnint akd (did) usgåas wis sin: walu asddnky (formation) n islmadn, walu idllisn (livres) n tivri. Avlif (ministère) n ulmud ur ismuttg (préparer) walu puma (afad) ad imnala (affronter, accueillir) d usgåas wis sin n uslmd n tmazivt. Avlif n ulmud islg (au sens d'"abandonner') ula min (mayd) tuva ivts (décider) ad iskr (igg) dg iyyar (igr) n uslmd n tmazivt mamc d yusa di tazmilt (note) 108. qao min tuva imirw (promettre) ad ygg dg usgåas wis sin idwl days (au sens de 'revenir sur') u islg it. Min ixs (ira) ad yini qao manaya? Ixs ad yini is imasayn (responsables) tirarn s ukaram (dossier) n uslmd n tmazivt. Ixs ad yini isur ur din bu ca n tanmlalt (bonnes intentions) bnid (arndad) i tmazivt d uslmd nns. Dix, Nzmmar ad nini is aslmd n tmazivt iggur (itdda) vur ubrir d avrud (échec complet). Maca tumayt (contenu) n abrir ad d tasrtit (politique) u maci d tabidgujit (pédagogique). Abrir ad iskan (iscan) d is aslmd n tmazivt ur illi vas (mvar) d tiknnaä d tixatarin n timaä (le plus grand leurre du siècle). Imala tiknnaä tixatarin n timaä wis 20 tuvat d "ääahir lbarbari", tiknnaä tixatarin n timaä wis 21 d aslmd n tmazivt. Tivtas (victimes) n tiknnaä ad d tmazivt u d imazivn. Isaän (héros) nns d inbbaän (ceux qui détiennent le pouvoir, ceux qui gouvernent) iorubyn. Ran ad ibrir (échouer) uslmd n tmazivt acku gådn ad yawi uslm n tmazivt var (vur) umarni (le fait de se développer, de croître) n ufrak (conscience) zi tnttit (identité) n tmazivt. Imala snsn (appliquer) avawas (plan) n usmtu (généralisation) n uslmd n tmazivt var qao iswirn (niveaux) n ulmud, ad trnu (vaincre) tmazivt taorabt. Ankam (conséquence) ad ur t ran inbbaän iorubyn acku nitni tddarn u nbän s taorabt u llan zzays. Imala tdda sn taorant ad ddan ula d nitni, ad tdda tanbaät nsn. Taorabt d amntil n tilawt nsn di lmuvrib. Zzays illan, zzays i nbän. Nitni, di tidtt, mmugrçn (regretter) aslmd n tmazivt. Umi ur zmmrn ad ssisrn (annuler) avtas n uslmd n tmazivt, qdcn (agir, faire) afad ad t ssbrirn. Ur ntettu is tadvrt (élite) n usinag vurs yat n tamasayt (responsabilité) d taxatart dg ubrir n uslmd n tmazivt. Asqsi i nsrusa d wad: is isul ad qqimn dg usinag awarni ubrir n uslmd n tmazivt? Imala suln dinni, min vad yili asmmigl (mission) nns dg uvarbuz ad? Acku imala ibrir uslmd n tmazivt ur d as itvimi min vad tskr. Ula d asinag ad ili isala asmmigl nns acku qao tiwuriwin n usinag tnnänt u tqqnnt var uslmd n tmazivt. Dix imala ibrir uslmd ad ur itvimi bu uszry (au sens de 'justification') i tilawt (existence) n usinag. 2002 Tawiza. .
