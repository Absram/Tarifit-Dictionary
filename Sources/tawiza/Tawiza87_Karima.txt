n tafukt (Bgayet) Tura t: KADA Karima (Dzayern Bgayt) Ugar n 50 n telmas n isserrifgen i yettidiren deg temnadvt n Iyer n Tafukt (Bgayet). Tilmas a snigirent s umata imukan tezdi temnadvt a. Tugett n telmas yeskanay d tiwtilin yelhan i tudert nnsent deg wemdiq a. Dga dayen ara yesseh bibren gef ismawen nnsent s Teqbaylit. Tazwart: Agama yezdi gar imgan d igersiwen. Ma yenger yiwen segsen, tawennadvt ad tenwiwel. Tamurt n Lezzayer, ttiddirent degs atttvas n telmas. Iswi n tezrawt yedran deg Iyer n Tafukt(Bgayet), d uddun n telmas n isserrifgen yettidiren degs. Tazrawt tella d gar useggas n 1998 d 1999. Gas akken alen drus, maca nessawedv ad nessekfel amur meqqren n isserrifgen. Tamnadvt n tezrawt: Amkan n tezrawt d Iyer n Tafukt d yiwet n taddart ibaoden s wazal n 65 Km gef temdint n Bgayet. Tezga d gef tigilt, gef tiddi n 950 m. Tamnadvt a tezdi ger tuddar n tgiwant n Kendira. Tin akken yellan deg tama n wenzul-asamar n bgayet . Akal is d azrar. Seg rrif n wasif Djemoa ar mi d tiqucac n idurar i d izzin i taddart a, nezmar ad nwali krad n tseddarin n yemyan. Azal n 276 n telmas n yemgan i yettwadden. Gar asent 76 d tid yettwasexdamen deg tujjya taorfant (KADA Karima, 2000). Tasnarayt: Tarrayt yettwadvefren deg tezrawt a d tin yebnan gef: - Asikked d umuqqel n isserrifgen srid s tit (tikwal s wal n usemger ma ara yili isserrifeg yebeed). - Ajerred n teskanin nnsen (ini, talga, ….asugu) - Acewwer n tudemt n isserrifeg. - Asteqsi neg tuttra n yemdanen yeddren attvas aya deg temnadvt nni. - I wakken ad naf isem ussnan n telmest, nettnadi deg yemniren n isserrifgen. Igemmad d tesladt: A :Awtem= mâle T :tawtemt=femelle I :oisillon Tasnisserrifegt= Ornithologie Isserrifeg= Oiseau Isifeg= Oisillon Igemmad yettwajerden deg tfelwit I, skanayen-d d akken azal n 50 n telmas yettidiren deg temnadvt n Iyer n Tafukt. Tagrayt: Igemmad n tezrawt yedran deg temnadvt n Iyer n Tafukt(Bgayet), gar useggas n 1998 d 1999, skanayen d akken azal n 50 n telmas n isserrifgen i yesnigiren degs. Ineggura ya bdan gef sin iremmueo, wid yettoeccicen deg wemdiq, s wazal n 90% d wid yettinigen am tfirellas, afmidi nnsen 10%.
