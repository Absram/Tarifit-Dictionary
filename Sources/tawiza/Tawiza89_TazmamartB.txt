Uriv yan n umgrad iffv d dg uïïun 38. Ass a (aujourd'hui), oawd umi saliv tivri n udllis n timsskta n oayda d ãalap pccad, uciv oawd s yan n uçuçä izzucl ayi ad ariv ca xf tazmamart. ittini yan n umdya: "imara (iv, mr) isla s umnär (Satan, diable) ad iqqn imjjan (imzzuvn) nns ". Ammu irvis (enfer) n tazmamart: anukkä (horreur) nns yujar min (may) innum itskr (itg) umn^är. Mqqar ntta d amnär itgr^ç (plaire) as ad isfld invmisn yxxun (être mauvais) xf iwdan, ur ivy (izmr) ad isl i unukkä n tazmamart acku yujar qao min issn xf inukkuän. Imala (imara) afgan (l'homme, awd) itmtta yat n tikklt (tawala) waha, di takurmut (prison) n tazmamart, itmtta aïïas n tikkal. Itmtta kuy as, kuy timrt (heure), kuy Tazmzt (instant). Vas (mvar) Di tazmamart, tamttant (la mort) ttavul zund (am) tudart (tudrt): itili vurs azmz, akud, avalal (durée). Suya (pour cela) Nzmmar ad nini is inkraf (prisonniers) n tazmamart mmutn 18 n isggusa, u maci ddarn 18 n isggusa di tazmamart. Acku tudart nsn tuva tt d tamttant d tamavlalt (permanente, continuelle). Ayad (manaya) iccan (iskan) ar mani tiw^^ä tasadit (sadisme) n inabaän (ceux qui gouvernent): ad nvn inkraf n tazmamart, urid ict n tikklt am qao imuddar (vivants) itmttan yat n tawala waha, ran ad tn nenvn kuy tazmzt, azdaw (durant) n 18 n isggusa. Dix, timgga (action de "faire") n umiär ur tlli d walu arndad (bnid) n timgga n inabaän nnv. Maca ur xarãn (penser) imiäirn i iskan (construire) tazmamart is takurmut a ad tetwassn var qao amaäal. Nitni vawsn (planifier) puma (pmak, bac) ad tqqim tazmamart d ufur (tagzi, secret), vad itwan^äln ééééé(êêéêêtre enterréééé) akd inkraf. suynni (pour cette raison) imasayn (responsables) d inabaän tuva ssban (nier) aliä (lbda) tilawt (existence) n takurmut n tazmamart. Maca aqqas (tuxsa, volonté) n tudart (tudrt) tarnu (vaincre) aqqas n inkba^än (criminels) u tssbrar (faire échouer) avaws (plan) nsn. Ddarn 18 zg inkraf n tazmamart. Ddarn puma (afad) ad ilin d inagan (témoins) xf unukkä. Ina (wid) d yavuln (dwln) zg irvis tnzganen (mériter) ad asn twaskan (être construit) isbdadn (statues) di kuy tmdint di lmuvrib. Acku zzaysn i nssn min (mad) imsarn di tazmamart. Mri (iv) ur zzaysn d ivul ula d ijjn, ili ommars (ursar) ad nssn ca xf irvis n tazmamart. Ili abkkaä (crime, anva) n inabaän d agrud (total) u imda (parfait). D nitni i danv issargbn (donner à voir, montrer) xf tidtt n inabaän nnv. Ammu i d ttffu (devenir) tazmamart d yan n ugmir (monument) n ukcuf (opprobre) dg umzruy asrti n lmuvrib anmtuta (contemporain). Tavul tazmamart ttdf dg usussn (tabadut, définition) n lmuvrib am (zund) ihrman (pyramides) n maãar (Égypte) tadfn (kccmn) dg usussn nns. Qao inni (winni) yurin xf tazmamart tarran tamasayt (responsabilité) var inaba^än waha. Maca tettun tamlilt (rôle, adduq) n ikabarn (partis politiques) i tuva tgnsisn (représenter) amnamar (opposition, tanmgla). Nitni, s ususm (silence) nsn xf ubkka^ä a axatar, ssdkn (appuyer, approuver) min (mad) skrn inabaän d isnnulufn d idaga^çn (tortures) d tikmawin (douleurs). Ikabarn ad ula d nitni d imasayn xf tikmawin ad acku (minzi) sqarn (fstn), ssusmn, ur zrurtn (désapprouver, désavouer) timkriäin (violations) n uzrf n ufgan (droits de l'homme). Da (vi) itwasars asqsi yad: imara ikabarn ad ur tmnamarn (s'opposer) ibkkaän n tnbaät (pouvoir), amnamar nsn itili d tirart waha ur vurs ula d yun n wammak (sens).
